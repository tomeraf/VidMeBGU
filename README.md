# BGU Moodle Video Downloader

Chrome extension to download videos from BGU Moodle course pages.

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable Developer mode in top-right corner
3. Click Load unpacked
4. Select this folder

## Usage

1. Navigate to any BGU Moodle video page
2. Wait 1-3 seconds - the extension icon will start animating with colors
3. Click the extension icon
4. Click the green Download Video button
5. Choose where to save the video

## Technical Notes

- Requires Chrome browser

## Privacy

This extension:
- Does not collect or transmit any data
- Does not track your browsing
- Runs entirely locally in your browser
